# 我的神经网络之旅，从pytorch开始手写神经网络  
## 一个普通大学生的神经网络实践代码  
### 1.下载依赖  
    conda create env -n net-learn python=3.10
    # 激活环境
    pip install -r requirements.txt

    #* 注意 *# ：请暂时注释掉pytorch相关，去官网下载适合自己硬件的pytorch

### 2.开始训练与评估  
* 分成了多个文件夹，每个文件夹都是一个小神经网络项目  
* 都包含较完整的训练代码，数据处理代码，可以按照代码内注释提示运行  
* 暂不公开数据集（太小太简单了，公开没意义，毕竟自己收集也是学习必要的[狗头]）
